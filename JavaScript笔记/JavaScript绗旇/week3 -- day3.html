堆栈内存

栈内存：（作用域）JS代码执行的地方，存放基本数据类型的地方

栈内存释放：

  • 全局作用域释放：关闭浏览器，释放当前全局栈内存
  • 私有作用域释放：
    • 一般情况下，函数执行结束就释放该栈内存，
    • 另一种情况下，不释放该栈内存

堆内存：浏览器开辟的内存空间，一般用来存放引用数据类型

堆内存的释放：将引用数据类型的变量名主动赋值null
目的：让引用内存空间没有变量来接收，浏览器会在空闲的时候将其释放
