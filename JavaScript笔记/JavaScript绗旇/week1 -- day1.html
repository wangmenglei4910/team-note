1 - JS的引入方式

- 外链式：页面js太多时 推荐使用 好维护
    <script src='./index.js'></script>
    注意：外链式和内嵌式不能混用
- 内嵌式
    <script type='text/javascript'>
        alert('内嵌式')
    </script>
- 行内式
    <h3 >你好</h3>

2 - JS中的变量和常量

变量：可以变化的量
  • 变量：可变化的值 存储数据
  • 定义变量：var
  • 语法：var 变量名 = 值
  • 关键字：在JS中有特殊意义 不能随便使用
  • = 表示赋值
  • 给变量修改值就是给变量重新赋值
常量：恒久不变的量
  • 常量：恒久不变的量 一旦被定义 就不会被修改
  • 定义常量：const
  • 语法：const 常量名 = 值
  • = 表示赋值
  • 注意：修改const定义的常量会报错

3 - JS的输出方式

  • 页面的输出方式：

  • document.write(text)： 将内容直接输出到body中 能够识别标签
  • innerHTML： 将内容输出到指定元素标签内
  • innerText： 将内容输出到指定元素标签内

innerHTML和innerText的区别：

相同点：

  • 将内容输出到指定的元素标签内

不同点：

  • innerHTML：可以识别内容中的html标签
  • innerText：不可以识别内容中的html标签 将内容当做普通文本 直接输出到页面指定的元素中

  • 弹框输出

  • alert('hello') 普通提示框
  • confirm("确定删除吗") 确认框
  • prompt 输入框

  • 控制台输出

  • console.log('控制台输出')
  • console.dir('在控制台详细输出')

4 - 变量的命名方式

  • 驼峰命名法(大驼峰 小驼峰)
  • 严格区分大小写
  • 数字 字母 下划线 美元符号$ 但是不能以数字开头
  • 不能使用关键字和保留字作为变量名

5 - 数据类型:基本数据类型 && 引用数据类型

  1. 基本数据类型

  • Nubner:数字类型 有效数字
  • String:字符串类型 单引号 双引号 包裹起来的都是字符串
  • Boolean:布尔类型 只有 true 和 false 两个值
  • Null null 空指针对象
  • Undefined：undefined 未定义
  • Symbel：唯一的值 ES6语法

  1. 引用数据类型

  • 对象数据类型
    • 普通对象 object {}
    • 数组 Array []
    • 正则 Regexp /\d+/
    • Data 日期对象
    • Math 数学工具
  • 函数数据类型
    • function 函数数据类型

6 - 检测数据类型

1-typeof

语法：typeof 值/变量

返回值：“number” “string” “undefined” “boolean” “object” “function”都是字符串包裹的数据类型

缺陷：不能够检测null和object，返回的结果都是“object”

通常用来检测基本数据类型和函数

2-instanceof

语法：实例 instanceof 类
含义：检测当前实例是否属于当前实例
返回值：true fasle
true 就表示属于当前类
false 就表示不属于当前类
