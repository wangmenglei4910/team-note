普通函数和构造函数的区别：

1，构造函数执行前 必须要加 new 操作符(构造函数如果不加new 就表示普通函数)
2，如果构造函数不需要传参数 那么可以省略小括号
3，构造函数中的this 指向当前实例
4，构造函数的执行步骤：

  • 1，形参赋值
  • 2，变量提升
  • 3，默认创建一个对象 让这个作用域下的this 指向这个对象
  • 4，代码自上而下执行
  • 5，把当前作用域下的this return出去
  • 6，作用域是否销毁

5，构造函数名为了和普通函数进行区分 构造函数名首字母一般大写
6，如果重写return值的：

  • 1，return一个基本数据类型的值 不能覆盖原有的实例
  • 2，如果return一个引用数据类型的值 会将默认返回的那个实例覆盖掉
