逻辑或

逻辑与

1.先看左边值 哪一个为假 就返回哪一个

3.如果都为false 返回第一个

break 和 continue

break

    // 只要在循环体中遇到break 整个循环就结束了
    // break后面的代码都会执行 以及i++
    
    for(var i = 0; i < 5; i++){
        if(i === 3){
            break;//一旦遇到berak 就结束整个循环
        }
        console.log(i)// 0 1 2 4 
    }

continue

    // 直接跳到下一轮 直接把continue忽略掉 但是不会中段代码
    for (var i = 0; i < 10; i++) {
      if (i < 5) { // i小于5的时候 遇到continue 
        // 不会执行continue后面的代码 直接跳到下一轮循环
        continue;
      }
      console.log('i', i) // 5 6 7 8 9
    }

获取页面元素

DOM提供document对象(文档对象)

1.根据class名获取元素

  • document.getElementById("ID名")
  • 通过ID名获取到的元素是一个对象

2.根据class名获取一个元素

  • document.getElementsByClass('类名')
  • 通过类名获取元素 返回的是个元素集合 获取到的元素对象都在集合里面
  • .length方法可以求出集合的长度

3.根据标签名获取一个元素

  • document.getElementsByTagName('标签名')
  • 通过标签名获取到的元素 返回的是一个元素集合 获取到的元素对象都在集合里面
  • .length可以求出集合的长度
