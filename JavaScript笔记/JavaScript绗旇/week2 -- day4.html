DOM的创建和添加

创建dom元素：createElement('小写标签名')

  • document.createElement('div')

添加/追加一个子节点appendChild('节点名称')

  • title.appendChild(h3)

插入一个子节点 insertBefore(新元素，参照元素)

  • box.insertBefore(h3,h2)
  
  // 创建元素
    var h3 = document.createElement('h3');
    // 给元素添加内容
    h3.innerHTML = '我是一个H3'
    console.log(h3)

    /*
        添加/追加一个子节点 appendChild();
        父容器.appendChild(子节点)
    */
    // 将h3元素追加到body底部
    document.body.appendChild(h3)

    // 添加在box里面的最后面
    box.appendChild(h3)

    // insertBefore
    // 插入到指定节点的前面
    // 父容器.insertBefore(新元素，参照元素)
    var h2 = box.children[0];
    box.insertBefore(h3,h2)

    /*
        js中的元素和页面中html标签 一一对应 映射关系(DOM映射)
        在页面中插入一个已有节点就是移动节点
    */

删除节点 父节点.removeChild('节点名称')

  • 有一个返回值：被删除的节点
  • list.removeChild('item1')

克隆节点:cloneNode(true/false)

  • true:可以克隆节点下面的子节点
  • false:不可以克隆节点的子节点

替换节点：replaceChild(新节点，旧节点)

  • list.removeChild(item11,item1)
     var list = document.getElementById('list')
    var item1 = list.children[0]

    var remove

    // 克隆节点
    var item11 = item1.cloneNode();
    console.log(item11)// 默认不能克隆item1的子孙节点 <li></li>
    item1.onclick = function(){
        console.log('我是item1')
    }

    var item12 = item1.cloneNode(true);
