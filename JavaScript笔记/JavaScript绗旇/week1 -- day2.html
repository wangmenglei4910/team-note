数字类型转换 Number

字符串转数字
  • 方法：Number(“String”) 强制转换
  • Number('') -> 0
  • Number("10px") -> NaN
  • 只要字符串中出现非有效字符，直接返回NaN
布尔类型转数字
  • Number(true) -> 1
  • Number(flase) -> 0
null 转数字
  • Number(null) -> 0
undefined
  • Number(undefined) -> NaN
数组 转数字
  • 先将数组转换成字符串，然后再转换成数字
  • Number([]) // Number([]) => Number('') => 0
  • Number([100]) // => Number('100') => 100
  • Number([10, 20, 30]) // Number('10, 20, 30') => NaN
  • Number([undefined]) => 0
对象 转数字
  • Number({}) => NaN
正则 转数字
  • Number(/^\d+$/) => NaN
函数 转数字
  • Number(function(){}) => NaN

数字转换 parseInt parseFloat

parseInt
  • 从左向右进行转换，一旦遇到非有效字符 停止转化 然后将转换的数字字符以数字形式返回
  • 如果左边第一个字符就是非有效字符，即直接返回NaN
  • 不能识别小数点 有取整数的作用 小数点后面的有效数字忽略不转换
    console.log(parseInt('A100')) // NaN
    console.log(parseInt('1A00')) // 1
    console.log(parseInt('10A0')) // 10
    console.log(parseInt('100A')) // 100
    // parseInt 不能识别小数点 它有取整作用
    parseInt(10.5) // 10
    parseInt('10.5px') // 10
parseFloat
  • 从左向右进行转换，一旦遇到非有效字符 停止转化 然后将转换的数字字符以数字形式返回
  • 如果左边第一个字符就是非有效字符，即直接返回NaN
  • 可以识别小数 小数点后面的数字也会进行转化
    parseFloat(10.5) // 10.5
    parseFloat('10.5px') // 10.5

isNaN 检测一个数是不是NaN

isNaN的返回值：

  • true:说明这个值不是有效数字
  • false:说明这个值是有效数字
  • 先将值使用Number方法转换，然后再进行判断
    // isNaN() 检测一个值是不是NaN(代表非数字)
    // isNaN(NaN) 如果是NaN 返回true 说明这个值不是有效数字
    // isNaN(1)  如果不是NaN 返回false 说明这个值是有效数字

    // isNaN()
    // 返回true 不是有效数字（NaN）
    // 返回false 是有效数字

    // isNaN(1 + undefined) // isNaN(1 + undefined) => isNaN(NaN) => true

    // isNaN先将值用Number()方法转换下 然后再判断是不是NaN
    isNaN('10') // isNaN(Number('10')) => isNaN(10) => false
    isNaN('10px') // isNaN(Number('10px')) => isNaN(NaN) => true

    isNaN(null) // isNaN(Number(null)) => isNaN(0) => false
    isNaN(true) // isNaN(Number(true)) => isNaN(1) => false
    isNaN([1]) // isNaN(Number([1])) => isNaN(1) => false

object 对象

创建一个对象

  • var 对象名 = {属性名:属性值}
  • 属性名:属性值 称之为 键值对 key:value
  • 键值对中间用逗号隔开

获取属性值的两种方式：

  • 对象名.属性名:不支持数字的属性名
  • 对象名['属性名']:属性名需要加引号，如果不加引号会把属性名当做一个变量来处理

增加属性

  • 对象名.属性名 = 值
  • 对象名[属性名] = 值

修改属性

  • 对象名.属性名 = 值

  • 对象名[属性名] = 值

  • 注意如果创建的对象里面有要修改这个属性名 就是修改这个属性名的属性值 如果创建的对象里面没有要修改的这个属性名 那么就是给当前对象增加一个属性

删除属性：

真删除:从当前对象里面删除这个属性
  • delete 对象名.属性名
  • delete 对象名[属性名]
假删除：在当前对象里面找到这个属性 重新赋值为null
  • 对象名.属性名 = null
  • 对象名[属性名] = null
